# OpenXPKI Installer
Simple bash script to install OpenXPKI on Debian

## How to Use
```bash
$ sudo su
$ git clone https://github.com/daffainfo/openxpki-installer
$ cd openxpki-installer
$ mv install.sh /root/
$ bash install.sh
```
After install, go to https://YOUR_IP/openxpki/

## Prerequisite
- Debian 10
